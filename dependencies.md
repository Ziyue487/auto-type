1. Install Homebrew
```python
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
2. Install Python
```python
brew install python3
```
3. Verify Python Installation:
```python
python3 --version
```
4. Install pip:
```python
pip3 --version
```
5. Upgrade pip:
```python
pip3 install --upgrade pip
```
